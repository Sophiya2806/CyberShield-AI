# 🛡️ CyberShield AI

AI-powered cybersecurity platform that analyzes security logs, detects potential threats, and provides intelligent security insights.

CyberShield AI helps identify suspicious activities, classify threats, and generate actionable recommendations using Machine Learning and secure backend architecture.

Built with React, FastAPI, Python, MongoDB, and Machine Learning.

Live Demo:
https://cyber-shield-ai-nine.vercel.app


## 🧠 About The Project

CyberShield AI is a full-stack cybersecurity platform designed to simplify security monitoring and threat analysis.

The system processes security logs, detects abnormal behavior, identifies possible attacks, and generates security reports for better decision-making.


## ✨ Features

- User authentication with JWT
- Secure password hashing
- Security log upload and analysis
- Support for Apache, Windows, Firewall, and authentication logs
- Machine Learning based threat detection
- Threat severity classification (LOW / MEDIUM / HIGH)
- AI-powered threat explanations
- Security dashboard with analytics
- Threat history tracking
- PDF security report generation
